# Start Page

The Start Page is the first thing you see when you open GetWrite — your home base across all your writing.

## Your projects at a glance

Every project you've created is listed here, along with running totals of how many projects and resources you've made. It's a quick way to take stock of your work and pick up where you left off.

## Working with projects

From the Start Page you can act on a project without opening it first:

- **Create** a new project — you'll choose a [project type](project-types.md) and give it a name.
- **Open** a project to start writing.
- **Rename** a project.
- **Delete** a project you no longer need.
- **Compile** a project into a single output. See [Compiling Projects](compiling.md).

Keeping these actions in one place means less time managing files and more time writing.
