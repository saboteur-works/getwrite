# Diff View

The Diff View compares two versions of a document and shows exactly what changed between them — what was added, what was removed, and what was modified.

Because every document keeps a [revision history](../revisions.md), you can look back at how a passage evolved instead of guessing. Did that paragraph get tighter or longer? What did you cut between drafts? The Diff View makes the answer precise, so you can revise with confidence and recover wording you thought was lost.
